# 🎯 QuizifyMe

**QuizifyMe** is a modern Flutter-based quiz application designed to make learning interactive and engaging. Students can participate in timed quizzes across multiple categories, compete on a leaderboard, track their progress, and enjoy premium features through a subscription system. Administrators can manage quiz content through a dedicated admin panel.

---

## 📱 Features

### 👨‍🎓 Student Module
- Secure student registration and login
- Firebase Authentication
- Timed quizzes with countdown timer
- Multiple quiz categories
- Automatic score calculation
- Personal profile with statistics
- Quiz history and total attempts
- Global leaderboard
- Dark & Light theme support
- Responsive and modern UI

### ⭐ Premium Subscription
- Mobile number verification
- OTP-based subscription
- BDApps Subscription API integration
- Subscription status checking
- Unsubscribe functionality
- Premium membership status

### 👨‍💼 Admin Module
- Secure admin authentication
- Quiz category management
- Question management
- Dashboard for content administration

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/elahi-999insidecell/QuizifyMe.git
```

### Navigate to the project

```bash
cd QuizifyMe
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
